Extensión de Cuadricep

MÁQUINA 1

Tarea

Realizar extension de cuadricep 12x3rep

Ejecución

Cómo hacer una extensión de cuádriceps correctamente

  1. Ajusta la máquina
  2. Coloca los pies bajo el rodillo
  3. Mantén la espalda recta
  4. Extiende las piernas
  5. Controla el descenso

Este ejercicio fortalece el cuádriceps y previene lesiones si se ejecuta con buena técnica.
